Useful informaiton:
- Run dev script = $ npm run dev
- Run build script = $ npm run build:prod (builds the dist folder which is uploaded to chrome://extensions)

To load the extension into chrome:
Navigate to chrome://extensions,
in the top right select developer mode,
in the top left click load unpacked,
select the 'dist' folder from the tuneOut folder
in the top right of chrome click on the puzzle piece icon,
click the pin next to the tuneOut application
Ready to go!
